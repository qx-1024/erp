# 库存管理模块表结构设计

## 1. 库存表 (inventory)

| 字段名 | 数据类型 | 长度 | 是否必填 | 默认值 | 描述 |
|-------|---------|------|---------|--------|------|
| id | BIGINT | - | 是 | 自增 | 库存ID |
| product_id | BIGINT | - | 是 | - | 商品ID |
| product_code | VARCHAR | 50 | 是 | - | 商品编码 |
| product_name | VARCHAR | 100 | 是 | - | 商品名称 |
| warehouse_id | BIGINT | - | 是 | - | 仓库ID |
| current_stock | INT | - | 是 | 0 | 当前库存 |
| available_stock | INT | - | 是 | 0 | 可用库存 |
| cost_price | DECIMAL | 10,2 | 是 | 0.00 | 成本价 |
| stock_amount | DECIMAL | 10,2 | 是 | 0.00 | 库存金额 |
| min_stock | INT | - | 否 | 0 | 最低库存 |
| max_stock | INT | - | 否 | 0 | 最高库存 |
| create_time | DATETIME | - | 是 | CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | - | 是 | CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

## 2. 库存流水表 (inventory_history)

| 字段名 | 数据类型 | 长度 | 是否必填 | 默认值 | 描述 |
|-------|---------|------|---------|--------|------|
| id | BIGINT | - | 是 | 自增 | 流水ID |
| product_id | BIGINT | - | 是 | - | 商品ID |
| product_code | VARCHAR | 50 | 是 | - | 商品编码 |
| product_name | VARCHAR | 100 | 是 | - | 商品名称 |
| warehouse_id | BIGINT | - | 是 | - | 仓库ID |
| type | VARCHAR | 20 | 是 | - | 操作类型(入库, 出库, 调拨, 盘点) |
| direction | TINYINT | - | 是 | - | 方向(1:增加, -1:减少) |
| quantity | INT | - | 是 | - | 数量 |
| unit_price | DECIMAL | 10,2 | 是 | 0.00 | 单价 |
| amount | DECIMAL | 10,2 | 是 | 0.00 | 金额 |
| balance | INT | - | 是 | - | 结存数量 |
| ref_id | VARCHAR | 50 | 否 | - | 关联单据ID |
| ref_type | VARCHAR | 20 | 否 | - | 关联单据类型 |
| operator | VARCHAR | 50 | 是 | - | 操作人 |
| create_time | DATETIME | - | 是 | CURRENT_TIMESTAMP | 创建时间 |

## 3. 库存盘点表 (inventory_check)

| 字段名 | 数据类型 | 长度 | 是否必填 | 默认值 | 描述 |
|-------|---------|------|---------|--------|------|
| id | BIGINT | - | 是 | 自增 | 盘点单ID |
| check_code | VARCHAR | 50 | 是 | - | 盘点单编号 |
| warehouse_id | BIGINT | - | 是 | - | 仓库ID |
| check_date | DATE | - | 是 | CURRENT_DATE | 盘点日期 |
| checker | VARCHAR | 50 | 是 | - | 盘点人 |
| status | VARCHAR | 20 | 是 | 待审核 | 状态(待审核, 已审核, 已完成) |
| total_amount | DECIMAL | 10,2 | 是 | 0.00 | 总金额 |
| difference_amount | DECIMAL | 10,2 | 是 | 0.00 | 差异金额 |
| remark | TEXT | - | 否 | - | 备注 |
| create_time | DATETIME | - | 是 | CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | - | 是 | CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |
