# 物流集成模块表结构设计

## 1. 物流表 (logistics)

| 字段名 | 数据类型 | 长度 | 是否必填 | 默认值 | 描述 |
|-------|---------|------|---------|--------|------|
| id | BIGINT | - | 是 | 自增 | 物流ID |
| logistics_id | VARCHAR | 50 | 是 | - | 物流编号 |
| order_id | BIGINT | - | 是 | - | 订单ID(采购订单ID/销售订单ID) |
| order_type | VARCHAR | 20 | 是 | - | 订单类型(采购订单, 销售订单) |
| logistics_company | VARCHAR | 50 | 是 | - | 物流公司 |
| logistics_no | VARCHAR | 100 | 是 | - | 物流单号 |
| logistics_status | VARCHAR | 50 | 是 | 待发货 | 物流状态(待发货, 已发货, 运输中, 已送达, 已签收) |
| sender_name | VARCHAR | 50 | 否 | - | 发件人姓名 |
| sender_phone | VARCHAR | 20 | 否 | - | 发件人电话 |
| sender_address | VARCHAR | 200 | 否 | - | 发件人地址 |
| receiver_name | VARCHAR | 50 | 否 | - | 收件人姓名 |
| receiver_phone | VARCHAR | 20 | 否 | - | 收件人电话 |
| receiver_address | VARCHAR | 200 | 否 | - | 收件人地址 |
| create_time | DATETIME | - | 是 | CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | - | 是 | CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |
| remark | VARCHAR | 200 | 否 | - | 备注 |

## 2. 物流轨迹表 (logistics_track)

| 字段名 | 数据类型 | 长度 | 是否必填 | 默认值 | 描述 |
|-------|---------|------|---------|--------|------|
| id | BIGINT | - | 是 | 自增 | 轨迹ID |
| logistics_id | BIGINT | - | 是 | - | 物流ID |
| logistics_no | VARCHAR | 100 | 是 | - | 物流单号 |
| status | VARCHAR | 50 | 是 | - | 状态 |
| location | VARCHAR | 100 | 否 | - | 位置 |
| description | TEXT | - | 否 | - | 描述 |
| track_time | DATETIME | - | 是 | CURRENT_TIMESTAMP | 轨迹时间 |
