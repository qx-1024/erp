# 多语言模块表结构设计

## 1. 语言表 (language)

| 字段名 | 数据类型 | 长度 | 是否必填 | 默认值 | 描述 |
|-------|---------|------|---------|--------|------|
| id | BIGINT | - | 是 | 自增 | 语言ID |
| language_code | VARCHAR | 10 | 是 | - | 语言代码(zh-CN, en-US, ja-JP等) |
| language_name | VARCHAR | 50 | 是 | - | 语言名称(简体中文, English, 日本語等) |
| status | TINYINT | - | 是 | 1 | 状态(1:启用, 0:禁用) |
| sort_order | INT | - | 否 | 0 | 排序序号 |
| create_time | DATETIME | - | 是 | CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | - | 是 | CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

## 2. 语言资源表 (language_resource)

| 字段名 | 数据类型 | 长度 | 是否必填 | 默认值 | 描述 |
|-------|---------|------|---------|--------|------|
| id | BIGINT | - | 是 | 自增 | 资源ID |
| resource_key | VARCHAR | 100 | 是 | - | 资源键 |
| language_code | VARCHAR | 10 | 是 | - | 语言代码 |
| resource_value | TEXT | - | 是 | - | 资源值 |
| create_time | DATETIME | - | 是 | CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | - | 是 | CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |
