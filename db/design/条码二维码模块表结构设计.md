# 条码/二维码模块表结构设计

## 1. 条码/二维码表 (barcode)

| 字段名 | 数据类型 | 长度 | 是否必填 | 默认值 | 描述 |
|-------|---------|------|---------|--------|------|
| id | BIGINT | - | 是 | 自增 | 条码/二维码ID |
| barcode_id | VARCHAR | 50 | 是 | - | 条码/二维码编号 |
| product_id | BIGINT | - | 是 | - | 商品ID |
| product_code | VARCHAR | 50 | 是 | - | 商品编码 |
| product_name | VARCHAR | 100 | 是 | - | 商品名称 |
| barcode_content | VARCHAR | 200 | 是 | - | 条码/二维码内容 |
| type | VARCHAR | 20 | 是 | - | 类型(条码, 二维码) |
| format | VARCHAR | 50 | 是 | - | 格式(EAN-13, Code 128, QR Code等) |
| generate_time | DATETIME | - | 是 | CURRENT_TIMESTAMP | 生成时间 |
| status | TINYINT | - | 是 | 1 | 状态(1:启用, 0:禁用) |
| remark | VARCHAR | 200 | 否 | - | 备注 |
