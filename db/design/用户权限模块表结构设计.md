# 用户权限模块表结构设计

## 1. 用户表 (t_user)

| 字段名 | 数据类型 | 长度 | 是否必填 | 默认值 | 描述 |
|-------|---------|------|---------|--------|------|
| id | BIGINT | - | 是 | 自增 | 用户ID |
| username | VARCHAR | 50 | 是 | - | 用户名 |
| password | VARCHAR | 100 | 是 | - | 密码（加密存储） |
| name | VARCHAR | 50 | 是 | - | 姓名 |
| email | VARCHAR | 50 | 否 | - | 邮箱 |
| phone | VARCHAR | 20 | 否 | - | 手机号 |
| language_preference | VARCHAR | 10 | 否 | zh-CN | 语言偏好 |
| status | TINYINT | - | 是 | 1 | 状态(1:启用, 0:禁用) |
| create_time | DATETIME | - | 是 | CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | - | 是 | CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

## 2. 角色表 (t_role)

| 字段名 | 数据类型 | 长度 | 是否必填 | 默认值 | 描述 |
|-------|---------|------|---------|--------|------|
| id | BIGINT | - | 是 | 自增 | 角色ID |
| role_code | VARCHAR | 50 | 是 | - | 角色编码 |
| role_name | VARCHAR | 50 | 是 | - | 角色名称 |
| description | TEXT | - | 否 | - | 角色描述 |
| status | TINYINT | - | 是 | 1 | 状态(1:启用, 0:禁用) |
| create_time | DATETIME | - | 是 | CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | - | 是 | CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

## 3. 权限表 (permission)

| 字段名 | 数据类型 | 长度 | 是否必填 | 默认值 | 描述 |
|-------|---------|------|---------|--------|------|
| id | BIGINT | - | 是 | 自增 | 权限ID |
| permission_code | VARCHAR | 50 | 是 | - | 权限编码 |
| permission_name | VARCHAR | 100 | 是 | - | 权限名称 |
| parent_id | BIGINT | - | 否 | NULL | 父权限ID |
| level | INT | - | 是 | 1 | 权限级别 |
| type | VARCHAR | 20 | 是 | - | 权限类型(菜单, 按钮) |
| path | VARCHAR | 200 | 否 | - | 权限路径 |
| component | VARCHAR | 200 | 否 | - | 组件路径 |
| icon | VARCHAR | 50 | 否 | - | 图标 |
| sort_order | INT | - | 否 | 0 | 排序序号 |
| status | TINYINT | - | 是 | 1 | 状态(1:启用, 0:禁用) |
| create_time | DATETIME | - | 是 | CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | - | 是 | CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

## 4. 用户角色关联表 (t_user_role)

| 字段名 | 数据类型 | 长度 | 是否必填 | 默认值 | 描述 |
|-------|---------|------|---------|--------|------|
| id | BIGINT | - | 是 | 自增 | 关联ID |
| user_id | BIGINT | - | 是 | - | 用户ID |
| role_id | BIGINT | - | 是 | - | 角色ID |
| create_time | DATETIME | - | 是 | CURRENT_TIMESTAMP | 创建时间 |

## 5. 角色权限关联表 (t_role_permission)

| 字段名 | 数据类型 | 长度 | 是否必填 | 默认值 | 描述 |
|-------|---------|------|---------|--------|------|
| id | BIGINT | - | 是 | 自增 | 关联ID |
| role_id | BIGINT | - | 是 | - | 角色ID |
| permission_id | BIGINT | - | 是 | - | 权限ID |
| create_time | DATETIME | - | 是 | CURRENT_TIMESTAMP | 创建时间 |
