# 销售管理模块表结构设计

## 1. 销售订单表 (sales_order)

| 字段名 | 数据类型 | 长度 | 是否必填 | 默认值 | 描述 |
|-------|---------|------|---------|--------|------|
| id | BIGINT | - | 是 | 自增 | 订单ID |
| order_code | VARCHAR | 50 | 是 | - | 订单编号 |
| customer_id | BIGINT | - | 是 | - | 客户ID |
| order_date | DATE | - | 是 | CURRENT_DATE | 订单日期 |
| expected_delivery_date | DATE | - | 否 | - | 预计交货日期 |
| logistics_company | VARCHAR | 50 | 否 | - | 物流公司 |
| logistics_no | VARCHAR | 100 | 否 | - | 物流单号 |
| logistics_status | VARCHAR | 50 | 否 | - | 物流状态 |
| status | VARCHAR | 20 | 是 | 待审核 | 状态(待审核, 已审核, 已取消, 已完成) |
| total_amount | DECIMAL | 10,2 | 是 | 0.00 | 总金额 |
| remark | TEXT | - | 否 | - | 备注 |
| create_time | DATETIME | - | 是 | CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | - | 是 | CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

## 2. 销售订单明细表 (sales_order_detail)

| 字段名 | 数据类型 | 长度 | 是否必填 | 默认值 | 描述 |
|-------|---------|------|---------|--------|------|
| id | BIGINT | - | 是 | 自增 | 明细ID |
| order_id | BIGINT | - | 是 | - | 销售订单ID |
| product_id | BIGINT | - | 是 | - | 商品ID |
| product_code | VARCHAR | 50 | 是 | - | 商品编码 |
| product_name | VARCHAR | 100 | 是 | - | 商品名称 |
| unit_id | BIGINT | - | 是 | - | 计量单位ID |
| quantity | INT | - | 是 | - | 销售数量 |
| unit_price | DECIMAL | 10,2 | 是 | 0.00 | 单价 |
| amount | DECIMAL | 10,2 | 是 | 0.00 | 金额 |
| remark | VARCHAR | 200 | 否 | - | 备注 |
