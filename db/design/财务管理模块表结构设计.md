# 财务管理模块表结构设计

## 1. 应收账款表 (accounts_receivable)

| 字段名 | 数据类型 | 长度 | 是否必填 | 默认值 | 描述 |
|-------|---------|------|---------|--------|------|
| id | BIGINT | - | 是 | 自增 | 应收账款ID |
| receivable_code | VARCHAR | 50 | 是 | - | 应收账款编号 |
| order_id | BIGINT | - | 是 | - | 销售订单ID |
| customer_id | BIGINT | - | 是 | - | 客户ID |
| amount | DECIMAL | 10,2 | 是 | 0.00 | 应收账款金额 |
| received_amount | DECIMAL | 10,2 | 是 | 0.00 | 已收金额 |
| balance | DECIMAL | 10,2 | 是 | 0.00 | 余额 |
| status | VARCHAR | 20 | 是 | 待收款 | 状态(待收款, 部分收款, 已结清) |
| due_date | DATE | - | 是 | - | 到期日期 |
| create_date | DATE | - | 是 | CURRENT_DATE | 创建日期 |
| remark | TEXT | - | 否 | - | 备注 |
| create_time | DATETIME | - | 是 | CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | - | 是 | CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

## 2. 应付账款表 (accounts_payable)

| 字段名 | 数据类型 | 长度 | 是否必填 | 默认值 | 描述 |
|-------|---------|------|---------|--------|------|
| id | BIGINT | - | 是 | 自增 | 应付账款ID |
| payable_code | VARCHAR | 50 | 是 | - | 应付账款编号 |
| order_id | BIGINT | - | 是 | - | 采购订单ID |
| supplier_id | BIGINT | - | 是 | - | 供应商ID |
| amount | DECIMAL | 10,2 | 是 | 0.00 | 应付账款金额 |
| paid_amount | DECIMAL | 10,2 | 是 | 0.00 | 已付金额 |
| balance | DECIMAL | 10,2 | 是 | 0.00 | 余额 |
| status | VARCHAR | 20 | 是 | 待付款 | 状态(待付款, 部分付款, 已结清) |
| due_date | DATE | - | 是 | - | 到期日期 |
| create_date | DATE | - | 是 | CURRENT_DATE | 创建日期 |
| remark | TEXT | - | 否 | - | 备注 |
| create_time | DATETIME | - | 是 | CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | - | 是 | CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

## 3. 费用表 (expense)

| 字段名 | 数据类型 | 长度 | 是否必填 | 默认值 | 描述 |
|-------|---------|------|---------|--------|------|
| id | BIGINT | - | 是 | 自增 | 费用ID |
| expense_code | VARCHAR | 50 | 是 | - | 费用编号 |
| expense_type | VARCHAR | 50 | 是 | - | 费用类型 |
| amount | DECIMAL | 10,2 | 是 | 0.00 | 费用金额 |
| applicant | VARCHAR | 50 | 是 | - | 申请人 |
| department | VARCHAR | 50 | 是 | - | 申请部门 |
| apply_date | DATE | - | 是 | CURRENT_DATE | 申请日期 |
| status | VARCHAR | 20 | 是 | 待审批 | 状态(待审批, 已审批, 已拒绝, 已报销) |
| remark | TEXT | - | 否 | - | 备注 |
| create_time | DATETIME | - | 是 | CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | - | 是 | CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

## 4. 收付款记录表 (payment_record)

| 字段名 | 数据类型 | 长度 | 是否必填 | 默认值 | 描述 |
|-------|---------|------|---------|--------|------|
| id | BIGINT | - | 是 | 自增 | 收付款记录ID |
| payment_code | VARCHAR | 50 | 是 | - | 收付款编号 |
| type | VARCHAR | 20 | 是 | - | 类型(收款, 付款) |
| business_id | BIGINT | - | 是 | - | 业务ID(应收账款ID/应付账款ID) |
| business_type | VARCHAR | 20 | 是 | - | 业务类型(应收账款, 应付账款) |
| amount | DECIMAL | 10,2 | 是 | 0.00 | 金额 |
| payment_method | VARCHAR | 50 | 是 | - | 付款方式 |
| payment_date | DATE | - | 是 | CURRENT_DATE | 收付款日期 |
| status | VARCHAR | 20 | 是 | 待审批 | 状态(待审批, 已审批, 已完成) |
| remark | TEXT | - | 否 | - | 备注 |
| create_time | DATETIME | - | 是 | CURRENT_TIMESTAMP | 创建时间 |
| update_time | DATETIME | - | 是 | CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |
